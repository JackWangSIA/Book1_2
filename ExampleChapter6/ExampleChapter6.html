EX 6-1 if-else语句

1.判断两个数组相等
A = [1 2 3]; B = [1 2 4];
if ~isequal(A,B) % 如果不相同
disp('A and B are not the same vector.')
end
A and B are not the same vector.
2.判断两个字符串相等
reply = 'yes';
if strcmp(reply,'yes')
disp('Your reply is "yes"!')
end
Your reply is "yes"!

EX 6-2 switch-case语句

order = 'food';
switch order
case 'food'
disp('OK, food is coming!')
case {'drink','fruit'}
disp('We do not have any.')
otherwise
disp('Have no idea.')
end
OK, food is coming!

EX 6-3 for语句

1.向量
vector = [1 2 4];
for i = vector
disp(i)
end
1 2 4
2.矩阵
matrix = rand(3,3)
matrix = 3×3
0.8147 0.9134 0.2785 0.9058 0.6324 0.5469 0.1270 0.0975 0.9575
for j = matrix
disp(j)
end
0.8147 0.9058 0.1270 0.9134 0.6324 0.0975 0.2785 0.5469 0.9575
3.字符
str = {'a','b','c'}
str = 1×3 cell 数组
'a' 'b' 'c'
for k = str
disp(k)
end
'a' 'b' 'c'

EX 6-5 break语句
limit = 0.8;
while 1
temp = rand;
if temp > limit
disp(temp)
break
end
end
0.9649
EX 6-6 continue语句
for n = 1:40
if mod(n,7)
continue
end
disp(['Divisible by 7: ' num2str(n)])
end
